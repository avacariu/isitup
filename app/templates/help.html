The help page

Hopefully this will clear up some things.

What is this?

isitup is a way to see which servers you've got are still running. It allows servers to call in to say that they're still running.

How do I use it?

  1. Click New thing in the menu.
  2. Give your new thing a name
  3. The Time to consider dead is the timedelta from the last call to determine if the server is dead.
  4. Hit Create
  5. Find your machine in the list and click the UUID
  6. Copy the Call home URL from the bottom of the page
  7. Add a cronjob that opens that URL

Tips

Use something like this as a cron job:

*/5 * * * * root curl --silent {{ config.SITE_URL }}/api/callhome/<UUID> > /dev/null
